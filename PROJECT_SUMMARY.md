# Café Site — Проект для портфолио

## 📋 Описание
Современный веб-сайт кафе «Тёплый Уголок» с полным функционалом для онлайн-заказов, админ-панелью и PWA-поддержкой.

## 🛠️ Технологический стек
- **Framework:** Next.js 14.2 (App Router)
- **Language:** JavaScript (ES6+)
- **Styling:** Tailwind CSS
- **State Management:** React Context API
- **Storage:** localStorage (корзина), Cookies (админ-сессия)
- **PWA:** Service Worker + Web App Manifest
- **Images:** Unsplash CDN + Next.js Image Optimization

## ✨ Ключевые функции

### 🍽️ Для посетителей
1. **Просмотр меню**
   - 12 блюд в 6 категориях (завтраки, супы, салаты, горячее, десерты, напитки)
   - Фильтрация по категориям
   - Поиск блюд по названию
   - Карточки с фото, описанием, ценой и весом

2. **Корзина заказов**
   - Добавление/удаление товаров
   - Управление количеством
   - Сохранение в localStorage (персистентность)
   - Подсчёт общей суммы
   - Плавающая кнопка (адаптивное позиционирование)

3. **Оформление заказа**
   - Форма клиента (имя, телефон)
   - Выбор типа: самовывоз / доставка
   - Отправка через WhatsApp или Telegram
   - Форматирование заказа в текстовый вид

4. **Страницы**
   - Главная (hero-секция, преимущества, популярные блюда, CTA)
   - Меню (полный каталог с фильтрами)
   - О кафе (история, галерея, преимущества)
   - Контакты (информация, карта, форма бронирования)

5. **PWA (Progressive Web App)**
   - Установка на устройство как приложение
   - Работа офлайн
   - Кэширование ресурсов через Service Worker
   - manifest.json с иконками и темой

### 👨‍💼 Админ-панель
1. **Авторизация**
   - Логин/пароль (admin / cafe2024)
   - HTTP-only cookies для сессии
   - Middleware для защиты маршрутов

2. **Дашборд**
   - Статистика (заказы, выручка, отзывы, блюда)
   - График выручки за неделю
   - Последние заказы (топ-5)
   - Последние отзывы (топ-3)

3. **Управление меню**
   - Просмотр всех блюд
   - Добавление новых позиций
   - Редактирование (название, цена, описание, категория)
   - Удаление блюд
   - Модальные окна подтверждения

4. **Управление заказами**
   - Список всех заказов
   - Фильтрация по статусу (новый, в работе, готов, доставлен)
   - Изменение статуса
   - Просмотр деталей заказа

5. **Управление отзывами**
   - Просмотр отзывов
   - Модерация (статусы: новый, опубликован, скрыт)
   - Удаление отзывов

## 🔧 Технические особенности

### 1. **Оптимизация**
- Статическая генерация (SSG) для всех страниц
- Lazy loading изображений
- Code splitting через Next.js
- Адаптивный дизайн (mobile-first)

### 2. **Состояние приложения**
- CartContext для глобального состояния корзины
- localStorage для персистентности
- React hooks (useState, useEffect, useContext, useRouter)

### 3. **Безопасность**
- HTTP-only cookies для админ-сессии
- Middleware для защиты маршрутов /admin/*
- Валидация на клиенте и сервере

### 4. **Доступность (A11y)**
- ARIA-атрибуты для кнопок и форм
- Семантическая вёрстка (header, main, footer, nav)
- Поддержка навигации с клавиатуры
- Контрастные цвета

### 5. **Анимации**
- Плавные переходы (hover, focus)
- Анимация появления (fade-up)
- Трансформации при скролле
- Кастомные скроллбары
